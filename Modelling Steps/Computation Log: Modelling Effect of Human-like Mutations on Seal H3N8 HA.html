Background:
With the new analysis of seal H3N8 released, I have decided to check the effect of the mutations that I suggested that Islam try out, based on an analysis of the receptor binding pocket.

Data:
I downloaded 4WA2 from the RSCB PDB, for the following reasons:
  1. It is the seal HA structure published in Yang et. al. (JVI, 2015)
  2. It has 3-SLN (Glyko® 3'-Sialyl-N-acetyllactosamine (3'-SLN),
    1. chemical form (copied from http://www.prozyme.com/product_details.html?key=GKAD-01014)
    3. This has an alpha(2,3) form.
FASTA file: 4WA2.fasta.txt
PDB file: 4WA2.pdb

I submitted all of these sequences to SwissModel for homology modeling using 4WA2 as the template file.

When searching for the mutations, note that the atom numbering changes.
  • 225 is now 217.
  • 226 is now 218.
  • 228 is now 220.
To show ligands, do S —> organic —> sticks

The current logic behind my current analysis is as such: If SwissModel is able to model in the ligand at the positions that are mutated, then that means there were no clashes between the sugar molecule and the amino acid side chains in the receptor binding site. This would indicate that the mutation would not be important for a receptor binding specificity switch from alpha(2,3) to alpha(2,6). Vice versa.

The mutations in 226 and 228 did not permit the sugar molecule to be modelled in. This suggests that they may allow for alpha(2,6) glycans in there, but I will need to test this by actual molecular docking.

Here we go with the new round of molecular docking! Based on the last round of molecular docking, I found that 3-SLN was not docked correctly into 4WA2. This may have to do with the overly-broad search space given to the program. I have narrowed the search space down, and this time, I have saved the configuration files used for each molecule.

I learned that the 4WA2-3SLN was not docked correctly, because I did not do the following:
  1. Remove ligand from 4WA2 molecule (use UCSF Chimera to do this).
  2. Remove solvent molecules (in UCSF Chimera)
  3. Made all ligand bonds non-rotatable (in Autodock Tools)

For the rest of the homology models of the single point mutants, I only had to do the following:
  1. Add polar hydrogens (in Autodock Tools)

Interpretation:
Binding affinity energies indicate how good the ligand will bind inside the designated receptor binding pocket. I first showed that the docking program will re-capitulate 3-SLN binding in 4WA2 properly, with a rigid ligand that does not rotate around any of its chemical bonds. With the same rigid configuration for 6-SLN obtained from 4KON, I show that it will bind more favourably to the Q226I-G228S mutated seal protein. As expected as well, single-mutated proteins have only intermediate binding affinities ~6.2-6.4 kcal/mol.
